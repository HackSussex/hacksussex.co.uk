GitHub

Where would tech be without GitHub? We highly recommend you use version control with your hack. Countless times teams lose hacks because of some problem or another. If you've never used GitHub before, a hack is a great place to learn!

Unity

Unity is a staple of the programming world. Download Unity for free and get started with any project from a game, to some machine learning or a mobile app.

Balsamiq

Balsamiq is a great app for creating wireframes for any project! Try it out with a free trial for your project today!

Chirp

Chirp.io is a superb communications project. Collect an API key in order to be able to send data through sound! Your data will be converted into a sound byte, and the receiving device will be able to reconstruct the data being sent!

Twilio

Twilio is a messaging service! Collect your free Twilio credits for your hack in order to be able to send texts programatically, spoof texts, host conference calls and many more features.

JetBrains

Big thanks to JetBrains, who are providing free JetBrains vouchers. Make sure you grab yours to download this must-have software for any developer!
